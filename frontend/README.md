# Quản Lý Dân Cư - Frontend (React + Vite)

## Yêu cầu
- Node.js 20+ và npm

## Chạy frontend (dev)
```bash
cd frontend
npm install
npm run dev
```

Mặc định Vite chạy tại `http://localhost:5173`.

## Kết nối backend
- Backend local cần chạy tại `http://localhost:8080`
- API được gọi theo prefix `/api` (ví dụ: `http://localhost:8080/api/...`)

## Build
```bash
npm run build
npm run preview
```
